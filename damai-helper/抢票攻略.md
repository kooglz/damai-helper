## 🎯 抢票战术

### 战前准备（开售前 1 小时）

```
□ 确认支付密码/指纹支付已设置
□ 检查观演人信息已添加到大麦账号
□ 测试网络速度（推荐用 5G WiFi 或有线）
□ 关闭其他占用带宽的应用
□ 手机 + 电脑同时准备
□ 大麦 APP 和网页端同时登录
```

### 抢票流程

| 时间 | 操作 |
|------|------|
| 开售前 10 分钟 | 打开倒计时，刷新页面 |
| 开售前 1 分钟 | 停止刷新，等待按钮变亮 |
| 开售前 10 秒 | 手指放在「立即购买」上方 |
| 开售瞬间 | 立即点击 → 选价位 → 选张数 → 提交 |
| 提交后 | 立即支付（不要犹豫） |

### 如果第一次失败

```
1. 不要退出！继续刷新
2. 5-15 分钟内会有未支付订单回流
3. 持续刷到开票后 30 分钟
4. 同时关注其他平台（猫眼、票星球）
```

## 🔥 回流票技巧

开票后持续刷新的原因：
1. **15 分钟回流**：未支付订单释放
2. **30 分钟回流**：部分锁单释放
3. **不定时回流**：系统释放的预留票

**最佳刷新策略：**
- 开售后前 5 分钟：疯狂刷新
- 5-15 分钟：每 3-5 秒刷新一次
- 15-30 分钟：持续刷新，有机会捡漏
