## 🚀 快速开始

### 方法一：Android Studio 构建（推荐）

1. **打开项目**
   ```bash
   # 在 Android Studio 中打开
   # File → Open → 选择 android-app 文件夹
   ```

2. **同步 Gradle**
   - 等待 Gradle 同步完成
   - 如有报错，检查 Android Studio 版本（建议 2022+）

3. **连接手机**
   - USB 调试模式开启
   - 或使用模拟器

4. **运行应用**
   - 点击 Run 按钮（绿色三角形）
   - 或使用快捷键 `Shift + F10`

### 方法二：命令行构建

```bash
cd ~/.openclaw/workspace/damai-helper/android-app

# 构建 Debug 版本
./gradlew assembleDebug

# APK 位置
# app/build/outputs/apk/debug/app-debug.apk
```

然后将 APK 传输到手机安装。

## ⚠️ 重要提示

### 必须手动完成的操作
1. **首次登录大麦账号** - 安全考虑，不能自动保存密码
2. **最终支付** - 提交订单后需手动完成支付
3. **开启无障碍权限** - 需要在系统设置中手动开启

### 提高成功率建议
1. **多设备准备** - 手机 + 电脑同时抢
2. **网络优化** - 使用稳定的 WiFi 或 5G
3. **清理后台** - 关闭其他应用
4. **支付准备** - 提前设置好指纹/面容支付
5. **电量充足** - 连接充电器

### 如果第一次失败
- 不要退出应用
- 继续刷新 15 分钟
- 会有未支付订单回流
- 5/10/15 分钟是关键时间点

## 🔧 故障排查

### 通知不响
1. 检查通知权限是否开启
2. 检查手机是否静音
3. 点击"测试通知"按钮验证

### 无障碍服务无法开启
1. 重启应用
2. 重启手机
3. 在设置中手动查找：设置 → 无障碍 → 已下载的服务

### 自动点击不生效
1. 确认无障碍服务已开启
2. 确认大麦 APP 已登录
3. 检查大麦 APP 版本（建议最新版）

### 应用闪退
1. 清理手机内存
2. 重启应用
3. 检查 Android 版本（需要 Android 7.0+）
