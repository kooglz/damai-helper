# 🎫 大麦抢票助手 - 孙燕姿苏州站

## 📦 文件说明

| 文件 | 用途 | 平台 |
|------|------|------|
| `damai-assistant.user.js` | 浏览器自动脚本 | 所有浏览器 + Tampermonkey |
| `remind.py` | 桌面倒计时提醒 | macOS / Linux |
| `抢票攻略.md` | 详细抢票策略 | 阅读参考 |

---

## 🚀 快速开始

### 方案 A：浏览器脚本（推荐）

1. 安装 Tampermonkey 扩展
2. 复制 `damai-assistant.user.js` 内容到 Tampermonkey
3. 打开大麦演出详情页
4. 看到倒计时浮窗即表示成功

### 方案 B：桌面提醒

```bash
cd ~/.openclaw/workspace/damai-helper
python3 remind.py
```

---

## ⚙️ 配置修改

**重要：** 使用前请修改以下配置：

### 浏览器脚本配置
找到 `CONFIG` 区域，修改：
```javascript
const CONFIG = {
    saleTime: '2026-03-13T10:00:00+08:00',  // 确认开票时间
    attendees: [
        { name: '你的姓名', idType: '身份证', idCard: '你的身份证号' }
    ],
    // ...
};
```

### 桌面提醒配置
编辑 `remind.py` 顶部 `CONFIG` 字典：
```python
CONFIG = {
    'sale_time': '2026-03-13 10:00:00',  # 确认开票时间
    # ...
}
```
